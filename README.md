# react-echarts-v3
React component wrap for ECharts(v3.x)


## Installation

```
$ npm install --save react-echarts-v3
```


## Usage

``` javascript
import IECharts from 'react-echarts-v3';

return (
    <IECharts option={option} />
);
```

## Properties

``` javascript
    option: React.PropTypes.object.isRequired,
    style:  React.PropTypes.object
```
